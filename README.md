# task-manager

## Бакалаврская работа на тему "Разработка клиент серверного приложения для управления задачами коммерческих предприятий"

Функционал:
- Создание компаний с гранулярными ролями
- Создание досок с настраиваемыми колонками
- Создание задач с комментариями и вложениями
- Получение уведомлений через почту и через SSE (Server-Sent Events)
